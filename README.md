# Jogo---Aprendiz-de-feiticeiro
Trabalho funal damateria de ED2 (estrutura de dados 2)

Este trabalho consiste em um jogo de rpg com arvore de habilidades, utilizando conceitos aprendidos em ED2 de arvore AVL. Eu fiz uma história principal extramamente simples e sem aprofundamento, para o usuário apenas tomar decisões e  adquirir pontos com elas, para assim conseguir comprar habilidades e personalizar a arvore da forma que preferir.
O jogador depois de cada escolha deita na história tem a opção de abrir o menu com as opções: ver habilidades, comprar habilidades, remover habilidades e sair do menu. Quando o uduário compra habilidades ele escolhe pelo id e insere na arvore - caso nescessário ela se autobalanceia - e apos isso sua quantidade de pontos é reduzida pelo valor gasto para obter a habilidade. Como ela é customizavel, a opção de remover habilidades pode um id da habilidade que deseja retirar (nó) e devolve o valor de pontos para o jogador e caso nescessário se auto balanceia.

Esse jogo é extremamente intuitivo e basta ler a história e as opções que aparecem no terminal para conseguir jogar e customizar suas habilidades e ficar cada vez mais forte.
